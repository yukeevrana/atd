# Circle is not right

## Синопсис

"*Circle is not right*" ("*Круг не прав*") - это казуальный инкрементальный tower-defence, разрабатываемый на движке Unity, с пост-ироничным сюжетом, отговаривающим игрока от игры за злого персонажа. Он создаётся в качестве первого завершённого (или наиболее близкого к завершённости) проекта, и служит частью портфолио для должностей: 

- Разработчик C#
- Разработчик Unity
- Геймдизайнер (баланс)
- Геймдизайнер (кор)
- Геймдизайнер (нарратив)

По каждому пункту далее будет указаны части проекта подробнее. Информация будет дополняться и обновляться со временем.

## Разработчик C#/Unity

В проекте используется код на языке C#, через который вызываются методы и инструменты Unity. Разработка ориентирована от кода к Unity, а не наоборот. Все скрипты с исходным кодом находятся в папке: `/Assets/Scripts`.

В проекте используются:

- Классы C# и объектно-ориентированный подход.
- Асинхронное создание игровых объектов с использованием пакета Addressables и async-await синтаксисом C#.
- Использование пула для переиспользования игровых объектов.
- Создание и использование префабов Unity.

## Геймдизайнер (баланс, кор, нарратив)

Для проекта пишется документация, в которой описан ход работы, сущности, механики и сюжет. Все файлы документации находятся в папке: `/Documentation`, и представляют собой текстовые файлы с markdown разметкой (как этот README файл). Для написания и чтения документации используется ПО Obsidian, но в силу широкого использования markdown разметки подойдет любой markdown просмотрщик или редактор, или вообще любой текстовый редактор, как VS Code или блокнот. 

- Кор механики взяты из схожих игр и доработаны под нужны проекта
- Баланс считается под нужды проекта
- Нарратив будет подан через диалоги, игровые механики и оповещения, и полностью описан в документации
