# Прототип 0.2.0

## Цель

Цель прототипа: добавить врагов и их уничтожение. 

Проект переосмыслен на быстрое получение результата, то-есть создание полноценного играбельного продукта, вместо тренировки и обучения сложным техникам.

Детально:

- Добавить появляющихся врагов
	- Враг - белый квадрат
	- У врага 1 единица здоровья
	- Враг появляется на окружности с регулируемым радиусом
	- Система спавна врагов использует пул объектов для врагов
	- Враг движется к центру экрана, пока не дойдёт или пока не упрётся в башню
- Добавить подсистему проверки столкновения снаряда с врагом
	- Расширить этой подсистемой систему стрельбы из пушки
	- Снаряд наносит 1 единицу урона
	- Снаряд исчезает после столкновения (становится неактивным)
