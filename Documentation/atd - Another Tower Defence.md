# Another Tower Defence

## Синопсис

- **Платформа**: Windows
- **Жанр**: Incremental Tower Defence
- **Сеттинг**: Геометрия
- **Движок**: Unity
- **Референсы**: 

Игра рассчитана, как первый законченный проект на Unity, поэтому имеет простейшую графику, простейший геймплей и простейшее звуковое сопровождение. Единственное место, где возможно будут приложены бóльшие усилия - сюжет, либо его не будет вовсе.

Главный герой - круг, стоящий в центре экрана. На него со всех сторон двигаются другие геометрические фигуры. Круг их атакует, за убийства получает валюту, а за валюту может улучшать оружие и броню.

## Геймплей

Уничтожение бегущих на башню врагов, улучшение башни.

## Взаимодействие

Клики по врагам и клики по элементам интерфейса улучшения башни.

## Сюжет и лор

Круг начинает немотивированный конфликт с другой геометрической фигурой, из-за чего они озлобляются на него и начинают атаковать. 

## Механики

Клики по врагам заставляют башню выстрелить по врагу. Улучшение башни через интерфейс улучшения улучшает её характеристики.

## Окружение

Единственный уровень бесконечен.

## Визуальный стиль

Простая геометрия.
