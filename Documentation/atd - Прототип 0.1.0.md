# Прототип 0.1.0

## Цель

Цель прототипа - сделать башню, которая может стрелять туда, куда кликнет игрок.

## Технические детали

Башня - белый круг в центре экрана. Снаряд - маленький белый круг, появляющийся на границе большого круга и летящий в направлении клика. Пропадает, отлетев на настраиваемое расстояние. Расстояние настраивается при помощи четырех значений - границ координат.

Использовать пул объектов для снарядов, то-есть создается новый снаряд, только если они закончились в пуле. При "пропадании" не удалять объект, а отключать и помещать в пул.
