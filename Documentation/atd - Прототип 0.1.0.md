# Прототип 0.1.0

## Цель

Цель прототипа - сделать башню, которая может стрелять туда, куда кликнет игрок.

## Технические детали

[[atd - Круг|Башня]] - белый круг в центре экрана. [[atd - Система стрельбы из пушки|Снаряд]] - маленький белый круг, появляющийся на границе большого круга и летящий в направлении клика. Пропадает, прожив определённое настраиваемое время.

Использовать пул объектов для снарядов, то-есть создается новый снаряд, только если они закончились в пуле. При "пропадании" не удалять объект, а отключать и помещать в пул.

## Результат

Прототип готов. Обновлены соответствующие файлы документации: [[atd - Круг]], [[atd - Система стрельбы из пушки]].
